# 一、基本知识

# 1.1前推后

  • 为了...，一定....
  • 只要 就
  • 所有 都
  • 。。。是。。。的充分条件

# 1.2否后推否前

例子1：

只要你买房，我就嫁给你。

前推后：买房 -> 嫁

否后推否前：不嫁 -> 不买房

例子2:

如果某人是东北人，那么一定是中国人。

东北人 -> 中国人

-中国人 -> -东北人

# 二、且和或

A且B：二者同时成立

A或B二者至少成立一个：包含三种情况， A ，B，AB都成立

要么A要么B：二者额只有一个成立。

# 2.1或的翻译：否1->1

或关系为真时，否定意向可以得到另一项为真
